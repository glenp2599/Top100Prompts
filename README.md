## Usage

Simply open `index.html` in your web browser to start exploring prompts. You can:

1. **Search** for prompts using the search bar
2. **Filter** by category using the dropdown menu
3. **Copy** any prompt with a single click
4. Browse through all 100 prompts organized in an easy-to-navigate grid

## Local Development

To run the website locally:

1. Clone this repository
2. Open `index.html` in your web browser
3. No build process or dependencies required!

Alternatively, you can use a simple HTTP server:
```bash
python3 -m http.server 8080
```

Then navigate to `http://localhost:8080` in your browser.
