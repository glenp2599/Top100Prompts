# Top 100 Network Engineer Commands

A comprehensive, interactive website featuring the top 100 essential commands every network engineer should know. This resource includes commands for Cisco IOS, Linux networking tools, troubleshooting, routing protocols, switching, security, and monitoring.

## Features

- **100 Essential Commands** - Carefully curated list of the most important network engineer commands
- **Category Filtering** - Filter commands by category (Cisco IOS, Linux, Troubleshooting, Routing, Switching, Security, Monitoring)
- **Real-time Search** - Instantly search through commands, titles, and descriptions
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Clean Interface** - Modern, professional design with easy-to-read command cards

## Usage

Simply open `index.html` in your web browser to access the website. You can:

1. **Browse All Commands** - Scroll through all 100 commands
2. **Search** - Type in the search box to find specific commands
3. **Filter by Category** - Click category buttons to view commands by type
4. **Combine Search and Filter** - Use both features together for precise results

## Categories

- **Cisco IOS** - Essential Cisco router and switch commands
- **Linux** - Linux networking and diagnostic tools
- **Troubleshooting** - Commands for diagnosing network issues
- **Routing** - Routing protocol commands (OSPF, EIGRP, BGP, etc.)
- **Switching** - Layer 2 switching commands (VLANs, trunking, etc.)
- **Security** - Security-related commands (ACLs, IPsec, authentication)
- **Monitoring** - Network monitoring and statistics commands

## Local Development

To run locally, you can use any web server. For example:

```bash
# Using Python 3
python3 -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js http-server
npx http-server
```

Then open `http://localhost:8000` in your browser.
